PHP 網頁範例 (使用 SNS COM+ V3.4)

系統需求  |  元件註冊方法 |  範例安裝與執行  | 原始碼

　

系統需求

   具備Apache+PHP+MySQL執行環境。本範例建置環境是至http://www.wampserver.com/en/下載wampserver3.2.0-x86做為PHP執行環境。

 

元件註冊方法

   需先註冊 SnsComServer_v3.4元件，註冊方法請參考 SNS_COM+ V3.4壓縮包有詳細COM 元件使用說明。

　

範例安裝與執行

1.          將SnsCom.php放至檔案目錄C:\wamp\www\下，再以瀏覽器開啟 http://localhost/SnsCom.php 測試。

2.          執行傳送訊息

n   目前SNS預設IP為”203.66.172.133”，Port為8001，必要時請設定成正確的IP和Port。

n   於SnsCom.php範例程式中，輸入您申請的帳號與密碼。
註：本範例是使用SnsComServer_v3.4.dll，參考的物件為 CHT.SNSCOMSERVER.SnsComObject3；

n   於網頁中，預設為傳送訊息的功能鈕，如圖1。

n   輸入手機號碼與訊息內容。

n   點選確定傳送按鈕，稍後即回應傳送訊息的結果。

3.          執行查詢狀態

n   目前SNS預設IP為”203.66.172.133”，Port為8001，必要時請設定成正確的IP和Port。

n   於SnsCom.php範例程式中，輸入您申請的帳號與密碼。

n   於網頁中，點選查詢狀態的功能鈕，如圖2。

n   輸入手機號碼與查詢Message_ID。

n   點選確定查詢按鈕，稍後即回應查詢狀態的結果。

n   目前SNS預設IP為”203.66.172.133”，Port為8001，必要時請設定成正確的IP和Port。

n   於SnsCom.php範例程式中，輸入您申請的帳號與密碼。

n   於網頁中，點選接收訊息的功能鈕，如圖3。

n   點選確定接收按鈕，稍後即接收訊息。

<?php

  

  //宣告一物件變數為SNSComServer元件

  $testCom=new COM("CHT.SNSCOMSERVER.SnsComObject3");  

 

  //預設登入IP為203.66.172.133, Port為8001

  $SNS_IP="203.66.172.133";

  $SNS_Port=8001;

 

  

  //輸入您的帳號和密碼

  $Account="";

  $Password="";

  

  //預設為傳送訊息的畫面

  $ShowSendForm="display:block";

  $ShowQryForm="display:none";

  $ShowRecvForm="display:none";

  //宣告變數的初始值

  $QryPhone="";

  $SendMsisdn="";

  $RecvMsisdn="";

  $RecvMsg="";

  

  $ResponseMsg="";  //傳送訊息或查詢狀態的結果

  if(count($_POST)>0){  

     $Func=$_POST["Func"];

  

  if ($Func == "1") {   //選擇傳送訊息的功能

     $SendPhone=$_POST["SendPhoneNum"];

     $Message=$_POST["Message"];

    

    //參數初始化

    $testCom->InitRemoteServer($_POST["IP"], $_POST["Port"], $_POST["Account"], $_POST["Password"]);

   

    //登入SNS Server

    $retCode_1 = $testCom->Login();

    

    //如果登入成功,才繼續執行傳送簡訊

    if ($retCode_1 = 1) {

        //傳送手機號碼, 訊息內容     

        $retCode_2 = $testCom-> SubmitMessage(null, $SendPhone, $Message, 1440);

        

        //依據回應碼判斷傳送結果

        switch ($retCode_2){

            case -1:

                $ResponseMsg=$testCom->GetRespDesc();

                break;

            case 0:

                $ResponseMsg="傳送成功;Message_id=" . $testCom->GetRespDesc();

                //查詢狀態中的手機號碼預設為傳送手機號碼

                $QryPhone = $SendPhone;

                //查詢狀態中的Message_Id預設為回傳的訊息，以備查詢

                 $MsgId = $testCom->GetRespDesc();

                 break;

            default:

                $ResponseMsg="傳送失敗(" . $retCode_2 . ");原因=" . $testCom->GetRespDesc();

        }

       

        //登出

        $testCom->Logout();   

//登入失敗

    }else{

       $ResponseMsg="登入失敗 :" . $testCom->GetRespDesc();

    }

    

    //控制在傳送訊息的畫面

    $ShowSendForm="display:block";       

    $ShowQryForm="display:none";

    $ShowRecvForm="display:none";

 

  }

 }
